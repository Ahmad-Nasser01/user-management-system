# User Management System

A full-stack web application designed for seamless user data management. It allows users to perform CRUD operations and status updates without reloading the page, powered by AJAX and jQuery.

## Features
- **Add Users:** Create new records for users with names and ages.
- **Display Data:** A clean, organized table view powered by Bootstrap.
- **Update Records:** Edit user details inline using an intuitive UI.
- **Toggle Status:** Quickly switch user status between 'Active' and 'Inactive'.
- **Delete Records:** Securely remove users with a confirmation prompt.
- **Dynamic UI:** Built with Bootstrap 5 for a responsive and modern design.

## Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5.
- **Backend:** PHP.
- **Database:** MySQL.
- **Communication:** jQuery & AJAX.

## Project Structure
- `index.php`: Main interface and client-side logic.
- `process.php`: Handles data retrieval and insertions.
- `db.php`: Database connection configuration.
- `toggle.php`, `update.php`, `delete.php`: Backend scripts for data manipulation.
